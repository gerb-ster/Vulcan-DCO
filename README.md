# Vulcan DCO

- 1v / oct control voltage
- -3v to 5v 8 oct control
    -3v = 0.0v = A0 = 27.5 Hz
    -2v = 0.5v = A1 =   55 Hz
    -1v = 1.0v = A2 =  110 Hz
     0v = 1.5v = A3 =  220 Hz
     1v = 2.0v = A4 =  440 Hz
     2v = 2.5v = A5 =  880 Hz
     3v = 3.0v = A6 = 1760 Hz
     4v = 3.5v = A7 = 3520 Hz
     5v = 4.0v = A8 = 7040 Hz
- 10vpp 1k impedance output
